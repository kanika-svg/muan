# Tonight in Vientiane — weekly post template

The weekly content engine for Muan. One curation run feeds three outputs:
the app (events.json), the TikTok carousel, and the Facebook post.
Post day: THURSDAY, 17:00–19:00 (people plan Friday on Thursday evening).

---

## Weekly workflow (target: under 2 hours)

1. **Collect (Wed night / Thu morning).** Scroll venue FB pages + local TikTok.
   Ask friends. Fill the spreadsheet: venue, event, date, time, price, source
   link, verified y/n.
2. **Verify.** Every event needs a source you actually saw (venue's own post).
   No source = not posted. If time/price unclear, message the venue page — they
   answer, and every reply is a relationship started.
3. **Update the app.** Copy verified events into data/events.json, commit
   ("Weekly events YYYY-MM-DD"), push. App is now current BEFORE the post
   goes out — the link must never disappoint.
4. **Make slides.** Canva template: 1 cover + 1 slide per event (5–8 events
   ideal). Real photo/clip per event; if none exists, venue exterior or a
   clean text slide. NEVER a wrong/unrelated photo.
5. **Post.** TikTok carousel + Facebook post, captions below. Same content,
   both platforms, same evening.
6. **Log.** Views / likes / shares / follows in the tracking sheet. This is
   the validation data that decides phase 2.

## Editorial rules (same as the app, same as LaoWander)

- No event without a checked source. Post fewer, correct, events.
- No superlatives as fact ("the best bar") — say what it is, not a ranking.
- Wrong info gets corrected loudly: comment + edit, never quiet deletion.
- No photo > wrong photo.
- If a venue asks to be included: yes, free, if real — that's the goal working.
  Mark nothing as "sponsored" unless money actually changed hands (none yet).

## Success bar (decided in advance — do not move the goalposts)

After 8 consecutive weekly posts (~2 months):
- ~300+ followers on either platform, OR
- consistent shares/saves from Vientiane locals, OR
- ≥2 venues have contacted us to be listed.
Any of these → build Phase 2 Slice A. None → rethink the angle before
writing backend code. Missing a Thursday resets nothing, but consistency
is the experiment — 8 posts, then judge.
